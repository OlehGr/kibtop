# Launching API

```
cd services/backend
docker compose up --build
```

**Api links:** \
Swagger - http://127.0.0.1:8000/api/v1/docs/ \
Redoc - http://127.0.0.1:8000/api/v1/docs/redoc/

**Admin link:** \
Django Admin - http://127.0.0.1:8000/admin/

# Launching React

```
cd services/frontend
docker compose up --build
```

**React link** - http://127.0.0.1:3000/
